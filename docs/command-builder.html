Generated command

zsh

 

Tip: Paste this into your terminal and press Enter. The runner connects through your local bridge at 127.0.0.1:11437. The bridge must have a Claude Code OAuth token available; it will not fall back to a real Anthropic API key. Use the playground folder for --session-id and the coordinator CLI. If the bridge says Unauthorized: Missing Bearer token, caller auth is enabled; either disable it in VS Code settings or set Caller auth token here. Use stream-json when another script or agent needs to parse every runner event (including compaction and stopReason). Set a human-readable log path when you want a plain text record of the prompt, tool calls, and final answer.
